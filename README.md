# 氢氦树 — 智慧树网课自动化辅助脚本

自动播放视频、处理课堂互动、切换下一节，带图形控制面板。

## 功能

| 功能 | 说明 |
|------|------|
| 自动播放视频 | 1.5 倍速播放，自动恢复暂停 |
| 课堂互动 | 自动检测并处理弹题（单选题/多选题/判断题） |
| 自动切换下一节 | 视频结束后自动进入后续章节 |
| 卡顿检测 | 25 秒无进度变化自动恢复 |
| 页面防检测 | 覆盖可见性 API + 随机人类行为模拟，最小化也不暂停 |
| 崩溃恢复 | 页面无响应时自动刷新，最多重试 3 次 |
| 控制面板 | 浏览器内浮窗，实时显示进度、统计、暂停/恢复，可拖拽可折叠 |

> 不处理单元检测、章节测验、考试等计分内容。

## 环境要求

- Node.js >= 16
- 操作系统：Windows / macOS / Linux

## 安装

```bash
# 先进入项目目录（将仓库克隆到哪里就进哪里）
cd <项目目录>

# 安装依赖
npm install
```

Playwright 会自动下载 Chromium 浏览器内核。如果安装失败，手动运行：

```bash
npx playwright install chromium
```

## 使用方法

### 方式一：attach 模式（推荐）

连接到本机的 Edge/Chrome，脚本会自动启动浏览器并打开智慧树。

```bash
npm start -- --attach
```

脚本会自动：
1. 杀掉残留的 Edge/Chrome 进程
2. 以调试模式重新启动浏览器，打开智慧树首页
3. 你在浏览器中登录并进入视频页面后，脚本自动接管

### 方式二：独立模式

脚本启动内置 Chromium 浏览器，需手动登录后进入视频页面。

```bash
npm start
```

在弹出的浏览器窗口中登录智慧树，进入任意课程视频即可。

### 方式三：无窗口模式

完全后台运行，不显示浏览器界面，配合 attach 使用效果最佳。

```bash
npm start -- --attach --headless
```

### 命令行选项

| 参数 | 说明 |
|------|------|
| `--attach` | 连接本机浏览器（自动启动 Edge/Chrome） |
| `--headless` | 无窗口后台运行 |
| `--debug` | 启用调试日志 |

## 控制面板

检测到视频后页面右下角出现浮窗面板：

- **深色主题**：暗青底色 `#0b1214` + `#09363d` 边框，微磨砂玻璃质感
- **标题栏**：左侧项目名称 + logo，右侧运行状态指示灯
- **暂停/恢复按钮**：运行态青色、暂停态淡青，带弹性过渡动画
- **统计数据**：播放累积时长、答题次数、当前倍速
- **进度条**：当前视频播放进度
- **折叠**：点击 `─` 折叠成迷你状态条，点击 `□` 展开
- **拖拽**：按住面板空白区域拖动到任意位置

## 配置说明

所有可调参数集中在 `config.js`，无需修改代码。

### 播放设置

```javascript
playbackRate: 1.5,      // 播放倍速（最高支持 2.0）
checkInterval: 2000,    // 检测间隔（毫秒）
```

### 答题坐标

```javascript
fixedCoords: {
  option: { x: 900, y: 570 },   // 选项点击位置
  close: { x: 919, y: 725 },    // 关闭按钮位置
  yRange: 30,                   // 选项二次点击时上下偏移范围(px)
  offset: { x: -80, y: -50 },   // 坐标修正偏移
}
```

> 坐标系以浏览器视口左上角为原点。截取答题弹窗后用测量工具确定坐标，在 `config.js` 中调整。

### 答题策略

检测到课堂互动题后：
1. 等待 3 秒（模拟思考）
2. 随机选择点击 **1 次**或 **3 次**（跳过 2 次，防止点掉已选中的选项）
3. 每次点击间隔固定 **1 秒**
4. 选项操作完成后等待 **2 秒**，再点击关闭

### 按钮文本匹配

```javascript
buttonTexts: {
  play: ['播放', '▶', '►', '继续播放'],
  next: ['下一节', '下一集', '下一个', '继续'],
  resume: ['继续学习', '继续播放', '从上次继续'],
  confirm: ['确认', '确定', '提交'],
  close: ['关闭', '关闭窗口', '×', '完成'],
  acknowledge: ['知道了', '好的'],
  speed: ['倍速', '速度', '速率'],
}
```

智慧树界面文字变化时在这里添加即可。

## 常见问题

### 播放倍速无效
智慧树最高支持 2.0 倍速。如果倍速切换失败，页面刷新后自动恢复。

### 点击位置不准
答题弹窗使用固定坐标点击，不同屏幕分辨率需调整 `config.js` 中 `fixedCoords`。

### 脚本卡在"等待视频页面..."
确认已在浏览器中登录智慧树并进入视频播放页面。

### 页面频繁刷新
检查网络连接是否稳定。脚本检测到页面崩溃时会自动刷新，最多重试 3 次。

### 最小化后视频会停吗？
不会。内置可见性覆盖，页面始终认为自己在前台。也可用 `--headless` 完全后台运行。
